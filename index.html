Getting Started

⚙️Installation

Requires Tampermonkey — a free browser extension for Chrome, Firefox, Edge, and Safari.

  1. Install the Tampermonkey extension for your browser.
  2. Open the Tampermonkey dashboard and click Create a new script.
  3. Delete the default content and paste in the full XariahTagger script.
  4. Press Ctrl + S to save.
  5. Navigate to xariah.net/eicons. The gold tag bar appears at the top, and a welcome message appears on your first visit.
🏷️

When XariahTagger is fully active, the browser tab icon changes to the XariahTagger logo. If it stays as the default Xariah icon, check that Tampermonkey's Allow user scripts browser permission is enabled in your browser's extension settings.

Core Feature

✏️Tagging Eicons

Hover over any eicon — a 🏷️ button appears in the bottom-left. Click it to open the tag editor.

  1. Hover over an eicon and click the 🏷️ button.
  2. Type a tag and press Enter or ,. It becomes a yellow chip.
  3. Add as many tags as you like, then click Save.

A gold glow on the 🏷️ button means the eicon already has tags. Tags are case-insensitive.

Suggestions — green, blue, and purple chips

Three rows of suggestion chips appear above the input as you work:

recent ↻
library match +
related ✦

Green — your 16 most recently used tags.
Blue — tags from your library matching what you're currently typing.
Purple — tags commonly found alongside the tags you've already added. These don't change when you type — they're based on your active tags, not your input.

Clicking any suggestion adds the tag and keeps your cursor in the field.

Copy Last and synonym auto-substitution

The ↩ Copy Last button loads all tags from your most recently saved eicon — useful when tagging a series of similar eicons. It won't overwrite a grid position if one is already set.

If you type a tag that has a synonym rule, it is silently replaced with the master tag and the new chip briefly flashes purple to signal the substitution.

What the yellow chip styles mean

Yellow chips reflect both save state and how the tag sits across your current selection:

nature ×
Solid gold — already saved, present on all selected eicons.
nature ×
Dim + dashed — added this session, not yet saved.
nature ×
Striped — saved, but not present on every eicon currently selected in preview.
nature ×
Striped + dim + dashed — added this session and only on some selected eicons.

A striped chip is called partial. It's only possible when you have multiple eicons selected in the preview panel — it tells you which tags aren't shared across the whole selection.

Productivity

☑️Batch Tagging & Selection

There are two ways to tag multiple eicons at once. Both end up in the same tag editor.

From the browse grid

Hold Ctrl and click eicons anywhere on the page — the main grid, search results, or Dev Tools. A floating gold bar appears at the bottom. Click 🏷️ Tag Selected to open the batch editor. Tags you add are merged in — existing tags on each eicon are preserved, and any tags already shared by all selected eicons are pre-loaded as yellow chips.

From inside the tag editor — group preview

When you open the tag editor for an eicon that belongs to a set, a preview panel appears above the modal showing all members of that set. From here you can work without leaving the editor:

  1. Click any cell in the preview to focus that eicon and edit its tags.
  2. Ctrl+click cells to build a selection across the group. The eicon you opened with is automatically included.
  3. With multiple cells selected, any tag you add is shared across all of them. Tags that don't exist on every selected eicon appear as striped (partial) chips.
  4. Click ⊞ Select all under the preview to select every member of the set at once.
  5. Press Save — all changes apply to all selected eicons in one go.
💡

The partial chip state is how you know a tag gap exists. If you see a striped chip, at least one selected eicon doesn't have that tag. Add the tag to cover everyone, or leave it as-is if the difference is intentional.

How tags are applied in batch

Tags added while all or most eicons are selected go into a shared layer — they apply to everyone. Tags added while only one eicon is focused go into that eicon's individual layer. On Save, each eicon receives the shared tags plus any of its own extras.

If you select eicons that belong to the same naming group (e.g. banner1, banner2, banner3), the batch modal will auto-detect the grid layout and assign positions to all of them automatically.

Marking linked sets with &&

With 2+ eicons selected, type && and press Enter to mark them as identical artwork. See Linked Sets for details.

Search

🔎Searching

Type any words — results rank by how closely tags match. Exact tag names rank first; words buried inside longer tags rank last. Multi-word searches score each word separately.

Quoted phrases

Wrap a phrase in quotes to match it as a verbatim tag. It appears as a ghost chip in the search bar and matches exactly — no fuzzy scoring.

"heart eyes"
sparkle

Click a ghost chip to edit it. Backspace on an empty input pops the last chip back to text. Use -"phrase" to exclude a phrase.

Negative terms & top tags

Prefix any word with - to exclude eicons tagged with it: nature -animated. Above the Best Matches row, the 5 most common tags in your current results appear as small chips — click one to add it to your search instantly.

Synonyms in search

If synonym rules are set up in Dev Tools, searching any term in a synonym group automatically finds eicons tagged with any related term. Synonyms work in both directions.

Advanced Feature

🔲Grid Tags & Eicon Sets

Multi-tile eicons — banners or panels made of several 100×100px eicons placed side by side — can be tagged as a set and copied as a complete BBCode block in one click.

Smart grid detection

When you tag an eicon that belongs to a detectable set, a Drag grid to size widget appears below the input. The ⚓ anchor is fixed at the top-left. Drag the red ↕ handle to adjust rows and the ↔ handle to adjust columns. Click Confirm ✓ to apply — positions are automatically assigned to all eicons in the set. You don't need to open each one individually.

Setting a grid tag manually

Type #rows×cols (e.g. #2x3) in the tag input and press Enter. A #2x3 × chip appears. A position picker appears below — click the square for this eicon's position. For sets larger than 5×5, manual entry is required.

As with the smart widget, typing a grid tag while on a detectable set will auto-assign positions to siblings.

⚓

When any set member appears in results, the full composite preview renders. Click it to copy all rows as BBCode. Ctrl+Click a composite to add it to your batch selection.

Forced grid groups (##)

Sometimes a grid is made of eicons with unrelated names — the smart detection won't pick them up automatically. You can force them into a group manually.

  1. Open the tag editor for the set. Ctrl+click all members in the preview panel.
  2. Type ## and press Enter.
  3. A grid size is assigned automatically. The ## chip appears left of the input whenever you're editing any member.

Forced groups are managed in 🛠️ Dev Tools → Forced Grids. Clicking the ## chip in the editor jumps there directly.

Deduplication

🔁Linked Sets (&&)

Some eicons are identical artwork uploaded under different filenames. Marking them as a linked set means only one appears per search result section.

  1. Ctrl+Click 2 or more eicons with identical artwork.
  2. Click 🏷️ Tag Selected in the floating bar.
  3. Type && and press Enter.

A && chip appears left of the input when editing any eicon in a linked set. Clicking it opens the Dev Tools → Duplicates tab and jumps to that set.

Content Control

🚫Blocked Tags

Any tag can be blocked. Eicons tagged with a blocked tag are censored everywhere — blurred, overlaid in grey, and framed with a hazard border. Works across the main page, search results, and Dev Tools.

Manage blocked tags in 🛠️ Dev Tools → 🚫 Blocked. Changes take effect immediately.

⚠️ Blocked tags are never exported

Blocked tags exist only in your browser. They are excluded from all exports and unaffected by imports — entirely personal.

Productivity

👁️Tag Mode

Click the red Tag Mode button to dim every already-tagged eicon, leaving only untagged ones at full visibility. As you tag and save, eicons fade out — making it easy to see what's left. Hover any dimmed eicon to restore full opacity temporarily. Click Tag Mode ON to deactivate.

Data

💾Import & Export

💾 exports everything — tags, grid positions, synonym rules, and linked sets — as a single xariah_tags.txt file. Export regularly; localStorage can be cleared by browser maintenance.

📥 imports a previously exported file. A dialog shows the contents before anything changes.

Merge

Adds new data from the file into your existing library. Nothing existing is removed. Recommended.

Replace

Replaces everything with the imported file. Cannot be undone without a prior backup.

🔒

Blocked tags are never included in exports and are never affected by imports.

Community

📡Developer Tag Library

The 📡 button fetches a curated tag library from the script's developer on GitHub and merges it into your local library. A confirmation shows exactly what will be added before anything changes. Always a Merge — your tags are never overwritten.

Power Feature

🛠️Dev Tools

Click 🛠️ to open the Dev Tools overlay. Close with ✕ or Escape.

📂 Tag Browser

The right panel lists every tag in your library as a coloured label. Sort by A→Z, Z→A, most eicons, or fewest using the buttons at the top. Filter the list live with the search input.

Click a tag to browse all eicons with that tag in the left panel. Ctrl+Click eicons to select them for batch tagging — a 🏷️ Tag Selected bar appears at the top of the panel when you have a selection.

🔗 Synonyms

Synonym rules make search terms interchangeable — searching happy also finds eicons tagged joy if they're linked. Rules are bidirectional.

To add: type the master tag, add slave tags by pressing Enter after each, then click Add Synonym Rule. To edit: click ✏️ Edit on any existing rule — it loads back into the form for modification.

⚠️ Clean Database

Optionally tick Clean database before adding to replace all slave tags with the master tag across your entire library. A confirmation shows how many eicons are affected. Cannot be undone without a backup.

🔁 Duplicates (Linked Sets)

Lists all linked sets as collapsible rows. Click a row to expand and see the eicons. Click 🗑 Delete Group to remove the link — eicons and their tags are unchanged.

Clicking the && chip in any tag editor jumps directly here and expands the relevant group.

📐 Forced Grids

Same interface as Duplicates — collapsible accordion rows, each showing the eicons in that forced group. Click 🗑 Delete Group to disband the group without affecting the eicons or their tags.

Clicking the ## chip in any tag editor jumps directly here.

🚫 Blocked Tags

Type a tag and press Enter or click Block Tag to add it to your block list. Active blocked tags are listed below with individual ✕ Unblock buttons. Changes apply immediately. Blocked tags are never exported.
