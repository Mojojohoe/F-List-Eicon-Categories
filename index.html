Introduction

❓What is XariahTagger?

Xariah.net is a database for searching and browsing the eicons used on the roleplaying platform F-List. Eicons can only be searched by their exact name — which is fine if you know the name, but hopeless if you just want to find "that blue sparkly one".

XariahTagger is a Tampermonkey browser script that layers a personal tagging system on top of Xariah. You give eicons searchable tags in plain English. You can then search those tags to instantly find what you're looking for.

Getting Started

⚙️Installation

XariahTagger requires Tampermonkey — a free browser extension that lets you run custom scripts on websites.

  1. Install the Tampermonkey extension for your browser (available for Chrome, Firefox, Edge, and Safari).
  2. Open the Tampermonkey dashboard and click Create a new script.
  3. Delete the default content and paste in the full XariahTagger script.
  4. Press Ctrl + S to save.
  5. Navigate to xariah.net — the gold tag bar should appear at the very top of the page.
Confirming the version

The tag bar shows a small version number in the top-left (e.g. v0.0.07). If you update the script and the version number doesn't change, Tampermonkey may be running a cached version — check that only one copy of the script exists in your dashboard.

The Search Input

Type any word into the search box. Results appear instantly as you type — no need to press Enter. The search checks every tag you've ever assigned across your entire library.

Best Matches

Above the category results, a gold-bordered ✨ Best Matches row appears. These are the eicons that match the most keywords in your search, ranked by relevance. Click any eicon to copy its BBCode immediately.

Category Results

Below Best Matches, each matching tag gets its own collapsible section. Click the section header to expand it and see all eicons tagged with that word. Each result can be clicked to copy its BBCode.

Collapse / Expand

The Collapse button hides the search results area, leaving only the slim header bar visible. This is useful when the bar is taking up too much vertical space. Click Expand to bring it back.

Core Feature

✏️Tagging an Eicon

Every eicon on Xariah shows a small 🏷️ tag button in the bottom-left corner when you hover over it. This is how you add and edit tags.

  1. Hover over any eicon on the Xariah browse page. A small tag button appears in the bottom-left corner.
  2. Click the 🏷️ button. The tag modal opens, showing the eicon's name at the top.
  3. Type a tag into the text field (e.g. nature, blue, sparkle).
  4. Press Enter or , to confirm the tag. It appears as a chip below.
  5. Repeat to add as many tags as you like.
  6. Click Save when done. The tag button on the eicon will glow gold to show it has been tagged.

Tag Chips

Confirmed tags appear as yellow chips. Click the × on any chip to remove that tag.

Blue chips are library suggestions — tags you've used on other eicons that match what you're currently typing. Handy for maintaining consistent tag names.

Copy Last

Once you've saved at least one eicon, a small ↩ Copy Last button appears in the top corner of the modal. Clicking it copies all tags from the most recently saved eicon into the current one — a fast way to tag multiple similar eicons in a row.

⚡

Tags are case-insensitive and stored in lowercase. "Nature", "NATURE", and "nature" all become the same tag.

Advanced Feature

🔲Grid Tags & Eicon Sets

Some eicons are designed as tiles in a larger image — a background scene or full-width banner made up of many 100×100px eicons placed side by side. XariahTagger can recognise these sets and copy the whole thing as correctly-formatted BBCode in one click.

Creating a Grid Tag

In the tag modal, type a grid tag in the format #rows×cols and press Enter. For example, a 2-row, 3-column set:

Grid tag format

#2x3

2 rows tall, 3 columns wide.

The grid tag appears as a red chip to the right of the text field:

#2x3 ×

A small grid appears below it. Click one of the squares to mark the position this eicon occupies in the set. The selected cell turns red.

●

Above: a 2×3 grid with the top-left position selected (row 1, col 1).

Do the same for every eicon in the set — assign each one its position. XariahTagger identifies which eicons belong together using their names (e.g. naturebg1, naturebg2, …).

Viewing the Set in Search Results

When any eicon from a tagged set appears in search results, XariahTagger automatically assembles the entire set as a composite preview — all tiles shown together in their correct positions. Empty positions are shown as dashed cells.

Copying the Set

Click anywhere on the composite preview to copy the full BBCode to your clipboard. Each row becomes one line, with empty positions filled using [eicon]none[/eicon] as a spacer so the layout stays correct:

[eicon]naturebg1[/eicon][eicon]naturebg2[/eicon][eicon]naturebg3[/eicon] [eicon]naturebg4[/eicon][eicon]none[/eicon][eicon]naturebg6[/eicon]

You can also hover over individual cells in the composite and click the 🏷️ button on any cell to edit that specific eicon's tags.

📐

Only one grid tag per eicon is allowed. Adding a second #NxM replaces the first. Clicking an already-selected cell deselects it.

Productivity Feature

👁️Tag Mode

When you're on a tagging session and want to focus only on eicons you haven't touched yet, Tag Mode is your friend.

Click the Tag Mode button in the header bar to activate it. Every eicon that already has tags attached becomes semi-transparent with a faint gold outline — visually pushed to the background.

Untagged eicons remain fully visible. As you tag them and save, they fade out immediately, leaving you a clear view of what still needs attention.

When Tag Mode is active

A red banner appears under the search bar reading "🔴 Tag Mode active — eicons with tags are hidden". Hover over any dimmed eicon to temporarily restore it to full opacity, and the tag button remains clickable.

Click Tag Mode ON again to deactivate — all eicons return to full visibility instantly.

Data Management

💾Import & Export

Your tags are stored in your browser's localStorage. To back them up, transfer them to another browser, or share them with someone else, use the Import and Export buttons in the top-right of the tag bar.

Exporting — 💾

Click the 💾 button. A file named xariah_tags.txt downloads immediately. This is a plain JSON file containing all your tags. Keep it somewhere safe as your backup.

Importing — 📥

Click the 📥 button and select a previously exported .txt file. A dialog appears asking how to handle the import:

Merge

Combines the imported tags with your existing ones. For eicons that appear in both files, the tag lists are unioned — new tags are added, existing tags are kept. Eicons only in your existing library are untouched. Recommended for most cases.

Replace

Completely discards your existing tags and replaces them with the imported file. Use with caution — this cannot be undone without a backup.

After either operation, the page reloads to apply the changes.

Search Feature

🚫Negative Search Terms

You can exclude results from your search by prefixing any word with a minus sign (-). Any eicon tagged with that word will be removed from all results — both Best Matches and category sections.

Example

nature -animated

Returns all eicons tagged with nature, but excludes any that are also tagged with animated.

You can chain multiple negative terms:

Multiple exclusions

blue -dark -sparkle

Returns eicons tagged blue that are not tagged dark and not tagged sparkle.

⚠️

Hyphens in tag names: the minus-prefix only applies when - is the very first character of a search word. A hyphen within a tag name (e.g. a tag called sci-fi) is unaffected. However, avoid starting a tag name with a hyphen — searching for it would be interpreted as an exclusion. If you have existing tags beginning with hyphens, rename them.

💡

A search of only negative terms (e.g. -animated -dark) returns no results — at least one positive word is required.

Best Practices

💡Tips & Tricks

Consistent tag vocabulary

The blue suggestion chips help with this — they show tags you've already used that match what you're typing. Try to use the same words for the same concepts (e.g. always nature instead of mixing nature, natural, and outdoors), so your search results stay clean.

Use multi-word searches

The Best Matches section scores eicons by how many of your search words appear in their tags. Searching blue sparkle magic will surface eicons tagged with all three before ones tagged with just one of them.

Tag first, search later

The more eicons you tag, the more useful the search becomes. A good tagging session with Tag Mode active — working through one category at a time — pays off immediately in search quality.

Export regularly

localStorage can be cleared by browser maintenance, privacy extensions, or clearing your browser data. Export your tags periodically so you never lose your work.

Gold glow = tagged

The 🏷️ button on each eicon glows gold if that eicon has tags. No glow means it's untagged. A quick way to spot what needs attention without turning on Tag Mode.
