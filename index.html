Introduction

❓What is XariahTagger?

Xariah.net is a database for F-List eicons. The problem: eicons can only be found by their exact filename. XariahTagger lets you add plain-English tags to eicons and search them by description instead.

📋

🔲

Duplicates

🛠️
Dev Tools
Browse by tag, clean synonyms, manage duplicate groups.
💡

Everything is stored locally in your browser. Nothing is sent to any server.

Getting Started

⚙️Installation

Requires Tampermonkey — a free browser extension for Chrome, Firefox, Edge, and Safari.

  1. Install the Tampermonkey extension for your browser.
  2. Open the Tampermonkey dashboard and click Create a new script.
  3. Delete the default content and paste in the full XariahTagger script.
  4. Press Ctrl + S to save.
  5. Navigate to xariah.net — the gold tag bar appears at the top of the page.
Confirming the version

The tag bar shows v0.0.19 in small text. If you don't see it after updating, check that only one copy of the script is enabled in Tampermonkey — two active scripts is a common trap.

How results are ordered

Results are ranked in four tiers per keyword. Exact tag matches rank highest; substrings buried inside longer words rank lowest:

The ✨ Best Matches row at the top shows the highest-scoring eicons. Below it, each matching tag gets its own collapsible section sorted by the same tier order — so "BAR" appears before "EMBARRASSED".

Core Feature

✏️Tagging Eicons

Hover over any eicon — a 🏷️ button appears in the bottom-left. Click it to open the tag editor.

  1. Hover over an eicon and click the 🏷️ button.
  2. Type a tag and press Enter or ,. It becomes a yellow chip.
  3. Add as many tags as you like, then click Save.

A gold glow on the 🏷️ button means the eicon has tags. Tags are case-insensitive — "Nature" and "nature" are the same tag.

Suggestions, Copy Last, and removing tags

As you type, suggestion chips appear above the input:

recently used ↻
library match +

Green — your 16 most recent tags. Blue — tags from your library matching what you're typing. Clicking either adds the tag and keeps your cursor in the field so you can keep typing or click more chips.

Remove any tag by clicking the × on its yellow chip. The ↩ Copy Last button (top-right of the modal) loads all tags from your most recently saved eicon — ideal for tagging similar eicons in sequence.

Productivity Feature

☑️Batch Tagging & Selection

Hold Ctrl and click multiple eicons to select them. A floating gold bar appears at the bottom of the screen.

💡

Selected eicons glow with a gold outline. Ctrl+Click works on eicons in search results and the Dev Tools Tag Browser too, not just the main browse grid. A plain click anywhere clears the selection.

Click 🏷️ Tag Selected in the floating bar. Tags you add here are merged into every selected eicon — their existing tags are preserved. Tags already shared by all selected eicons appear pre-loaded as yellow chips.

Marking duplicates with &&

With 2+ eicons selected in the batch modal, type && and press Enter to mark them as duplicates of each other. See Duplicate Eicons for the full explanation of what this does.

Search

🔎Searching & Negative Terms

Type any word to search. Multi-word searches score each word separately — eicons matching more words rank higher. Prefix a word with - to exclude results tagged with it.

Negative search example

nature -animated

Shows eicons tagged nature that are not tagged animated. Multiple exclusions work: nature -animated -dark

⚠️

The - exclusion only applies at the very start of a search word. A tag like sci-fi is unaffected. Avoid naming tags starting with a hyphen.

Synonyms in search

If synonym rules exist (set up in Dev Tools), searching any term in a synonym group automatically includes the whole group. Searching happy also surfaces eicons tagged joy if those are linked. Synonyms are bidirectional — either end finds everything.

Advanced Feature

🔲Grid Tags & Eicon Sets

Many eicons are tiles in a larger image — a 2×3 banner made of six 100×100px eicons. XariahTagger can recognise these sets and copy the complete BBCode in one click, including [eicon]none[/eicon] spacers for any empty positions.

Creating a grid tag

In the tag editor, type #rows×cols (e.g. #2x3) and press Enter. A red chip appears. A small grid appears below it — click the square for this eicon's position in the set. Repeat for every eicon in the group.

●

When any set member appears in search results, the full composite preview is shown. Click to copy all rows.

How eicons are grouped automatically

XariahTagger strips trailing numbers or short suffixes and groups eicons sharing the same base. naturebg1, naturebg2, naturebg3 → base naturebg. Bases under 3 characters are never grouped. The && duplicate feature is a separate concept — see below.

Deduplication

🔁Duplicate Eicons (&&)

Some eicons are identical artwork uploaded under different names. Left unmanaged, every version appears in search results, wasting space. The && tag marks them as a group so only one appears per result section.

Creating a duplicate group

  1. Ctrl+Click 2 or more eicons with identical artwork to select them.
  2. Click 🏷️ Tag Selected in the floating bar.
  3. Type && and press Enter. The group is saved.

The && chip

When you open the tag editor for an eicon in a duplicate group, a && chip appears to the left of the input. It cannot be removed from here. Clicking it opens the Dev Tools → Duplicates tab and scrolls directly to that eicon's group with a gold flash so you can manage or delete it.

How deduplication works in results

Within each search result section, only the alphabetically first member of a group that exists in your library is shown. All others are silently skipped. The same artwork occupies exactly one slot per category regardless of how many uploaded names it has.

Productivity Feature

👁️Tag Mode

Click the red Tag Mode button to dim every already-tagged eicon, leaving only untagged ones fully visible. As you tag and save, eicons fade out immediately — giving you a clear view of what still needs work.

Hover over any dimmed eicon to restore full opacity temporarily. The 🏷️ button remains clickable. Click Tag Mode ON to deactivate.

Data Management

💾Import & Export

The 💾 button exports your entire library as xariah_tags.txt — one file containing tags, grid positions, synonym rules, and duplicate groups. Keep a copy; localStorage can be cleared by browser maintenance.

The 📥 button imports a previously exported file. A dialog shows the contents before anything changes:

Merge

Adds what's new from the imported file into your existing data. New tags, new synonym rules, new duplicate groups — all added. Nothing existing is ever removed. Recommended for most cases.

Replace

Discards all existing tags, grids, synonyms, and duplicates and replaces them entirely with the imported file. Cannot be undone without a backup.

Community Feature

📡Developer Tag Library

The 📡 button fetches a curated tag library from the script's developer on GitHub and merges it into your local library. This is a one-way pull — your data never leaves your browser, and you can always cancel before anything is committed.

A confirmation shows exactly how many new eicons and tags will be added. It is always a Merge — your own tags are never overwritten.

📡

The button shows ⏳ while fetching. If it fails (offline, server error), an error appears and nothing changes.

Power Feature

🛠️Dev Tools Window

The 🛠️ button opens a full-screen overlay with three tabs. Close it with ✕ or press Escape.

📂 Tag Browser — browse your library visually

The right panel lists every tag in your library with a count of how many eicons use it. Click a tag to populate the left panel with all eicons using it, shown as a masonry grid identical to search results — composites and all.

The filter input narrows the tag list live. Ctrl+Click eicons in the left panel to add them to your batch selection (the floating bar stays in sync). Hovering shows name tooltips; clicking 🏷️ opens the tag editor.

🔗 Synonyms — make search terms interchangeable

Synonym rules link tags together in search. Define happy as the master with slaves joy and cheerful — searching any of those three words finds eicons tagged with any of them.

To create a rule: type the master tag, add slaves by typing + Enter, then click Add Synonym Rule.

⚠️ Clean Database option

Checking Clean database before adding will replace every instance of the slave tags with the master tag across your entire library. A confirmation shows exactly how many eicons are affected. This cannot be undone without a backup export. The synonym rule is saved regardless.

Active rules are listed below. Click 🗑 Delete to remove a rule — this removes the synonym mapping only, not any previous clean.

🔁 Duplicates — manage duplicate eicon groups

Lists all duplicate groups. Each group is a collapsible row. Click a row header to expand it and see the eicons inside as a grid. Click 🗑 Delete Group to remove the grouping — the eicons and their tags are unchanged.

When you click the && chip in any tag editor, you jump directly here and the relevant group is expanded and scrolled into view with a brief gold highlight.
